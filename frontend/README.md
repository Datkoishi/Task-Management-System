# Frontend - Task Management System

## Cài đặt

1. Cài đặt dependencies:
```bash
npm install
```

2. Chạy development server:
```bash
npm run dev
```

Ứng dụng sẽ chạy tại `http://localhost:3000`

## Build

Để build production:
```bash
npm run build
```

## Công nghệ sử dụng

- React 18
- React Router DOM
- Axios
- Tailwind CSS
- Recharts (biểu đồ)
- date-fns (xử lý ngày tháng)
- Vite (build tool)

## Tính năng

- Đăng ký / Đăng nhập
- Dashboard với thống kê và biểu đồ
- Quản lý nhiệm vụ (CRUD)
- Checklist tự động cập nhật trạng thái
- Gán nhiệm vụ cho nhiều người
- Xuất báo cáo (CSV, Excel, PDF)
- Quản trị viên có thể quản lý users và xem tất cả tasks
- Responsive design
